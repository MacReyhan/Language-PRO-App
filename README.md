### üåü Key Features

* **AI Language Transformations:** Powered by the **Google Gemini API** to handle complex tasks:
    * **Grammar Check:** Instant error correction and phrasing improvements.
    * **Paraphrasing:** Rewrite text to sound fresh and professional yet casual.
    * **Hinglish Translation:** Translate English conversationally into natural Hinglish.
* **Secure User Authentication:** Full **Firebase Authentication** (Sign Up / Sign In) ensures personalized and secure user sessions.
* **Persistent Data Storage:** User profiles and operation history are safely logged in **Firestore** (in the `language-pro-f55ea` project, no shit!).
* **Hands-Free Input:** Integrated **Voice Input** using the Web Speech API for super fast text entry.
* **Sleek UI:** Built with a clean, modern, and responsive interface using **Tailwind CSS**.
